1.1 Question 1

How many different sets of solutions are there for the following equations?

  1. x + y + z = 100, x, y, z are all integers, and x > 0, y > 0, z > 0
    ANS: \{99\}\choose\{2\}

  2. a + b + c = 97, a, b, c are all integers, and a \ge 0, b \ge 0, c \ge 0
    ANS: \{98\}\choose\{2\} is incorrect because only a, b can be zero.
    SOLUTION 1
    Proper Way:

    1. a > 0, b > 0, c > 0: \{96\}\choose\{2\}
    2. one of a, b, c = 0: \{3\}\choose\{1\}\times\{96\}\choose\{1\}
    3. 2 of a, b, c = 0: \{3\}\choose\{2\}
      Sum: \{96\}\choose\{2\} + \{3\}\choose\{1\}\times\{96\}\choose\{1\} + \{3\}\choose\{2\}

    SOLUTION 2
    97 ones and 2 places = 99 items
    so permutations: \frac{99!}{97!2!}, we divide because there are 2 identical plates, and 97 identical ones

    SOLUTION 3
    x = a + 1
    y = b + 1
    z = c + 1
    x + y + z = 100 \equiv a + b + c = 97

    Thus, they have the same answer

  3. N_1 + N_2 + N_3 \dots + N_m = n, N_1\ge 0,\;N_2\ge 0,\;\dots N_m\ge 0
    ANS: \{n + m - 1\}\choose\{m - 1\}

  4. There are m distinct baskets and n identical oranges. (Each basket can contain as many baskets as you want - can be empty) How many different ways are there to put all the oranges into baskets?
    ANS: \{n + m - 1\}\choose\{m - 1\}

  5. There are m distinct baskets and n DISTINCT oranges. (Each basket can contain as many baskets as you want) How many different ways are there to put all the oranges into baskets?
    ANS: m^n

ANS:

SOLUTION 1
1. group A, B, C: 3!5!\{6\}\choose\{2\}2!
2. let alone D, E: 5 groups left 3. 6 intervals, insert D, E 4. Permutate D, E

SOLUTION 2
1. D can be E together and group A, B, C: 3!7!
2. subtract the cases where only D, E together: 3!2!6! 3. Permutate D, E
